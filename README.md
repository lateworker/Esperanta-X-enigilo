# Esperanta X-enigilo

A tiny program that can automatically convert "gx" into one letter "ĝ" in eo, driven by AutoHotkey.

Malgranda programo, kiu aŭtomate konvertas "gx" al la litero "ĝ" en Esperanto, funkciigita per AutoHotkey.

一个将 gx 转为字母 ĝ 的世界语输入工具。

# Kiel uzi（使用方法）

Post lanĉo, la programo tuj aperas kiel piktogramo en la sistempleto. Ŝvebante per la muso super la piktogramo, vi povas vidi la nunan staton. Dekstre alklakante ĝin, malfermiĝas kompleta agorda menuo, kie haveblas ĉiuj agordaj opcioj. Inter ili, vi povas laŭbezone ĝustigi la tempan sojlon, kiu difinas la maksimuman intertempon inter du klavpremoj por ekigi aŭtomatan anstataŭigon (defaŭlte: 100 ms).

程序运行后立即进入托盘图标，鼠标悬停于图标上可查看当前状态，右键托盘图标可以看到全部设置项，可自行调整触发时间阈值（默认100ms）
